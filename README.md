# biblioteca-Digital-mongoDB

Projeto simples desenvolvido em Node.js utilizando MongoDB Atlas, simulando uma biblioteca digital para armazenar livros, capas e conteúdos relacionados.

    Tecnologias Utilizadas

Node.js
MongoDB Atlas
MongoDB Driver
Dotenv
JavaScript


    Funcionalidades

Conexão com banco de dados MongoDB ☁️
Inserção de livros 📖
Estrutura de dados flexível (NoSQL)
Simulação de uma biblioteca digital

    Conceitos Aplicados

Banco de dados NoSQL
Estrutura baseada em documentos (JSON)
Conexão com banco na nuvem
Uso de variáveis de ambiente
Programação assíncrona (async/await)

    Questões Norteadoras

    Quais entidades e relacionamentos seriam difíceis de modelar apenas com tabelas?
Dados com estrutura variável (ex: livros com campos diferentes)
Relacionamentos complexos (autor, categorias, mídias)
Conteúdos multimídia (imagens, vídeos)

    Como armazenar dados hierárquicos?
Utilizando documentos aninhados (JSON)
Exemplo: um livro pode conter um array de capítulos dentro dele

    Que tipo de consultas seriam complexas ou lentas?
Pesquisar um livro que não está, não pesquisar algo em especifico
